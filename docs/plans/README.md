# Plans

Use plans only when work spans multiple sessions or needs sequencing.

## Locations

- `active/`: current work
- `archive/`: done or stale work

## Plan Shape

- Goal
- Scope
- Decisions
- Next

## Rules

- Keep each plan under 80 lines.
- Link to source files instead of copying large context blocks.
- Do not create a plan for small, one-session edits.
