# Project Overview

## Repo

- This repository is a generic Next.js starter.
- It is meant to be cloned, renamed, and customized with `starter:init`.
- The current home page and metadata are placeholder starter surface area.

## Stable

- Next.js 16 App Router
- Tailwind v4 theme tokens in `app/globals.css`
- shadcn primitives in `components/ui/`
- `cn()` helper in `lib/utils.ts`
- Geist fonts via `app/fonts.ts`
- Starter bootstrap in `scripts/init-starter.mjs`

## Notes

- Keep the repo starter-only. No product-specific flows or secrets.
- `README.md`, `package.json`, `.env.example`, and starter files are the main
  customization surfaces.
